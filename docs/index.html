FAQ

Is this safe to use? +

Yes. Built-in risk limits prevent catastrophic losses. Start with paper trading (no real money) to test strategies. When live, you control max position sizes, daily loss limits, and which markets the bot can trade.

How much does it cost? +

ProbablyProfit is free and open source. You pay only for AI API usage (typically $0.01-0.05 per market analysis). No platform fees, no profit sharing, no subscriptions.

What markets can it trade? +

Currently supports Polymarket (crypto, requires VPN in US) and Kalshi (regulated US exchange). Both are binary prediction markets where you bet on real-world events.

Will this make me money? +

No guarantees. Prediction markets are competitive. AI helps analyze faster, but edge comes from your strategy. Backtest thoroughly, start small, and never trade money you can't afford to lose.

Or use the Python API

python 
from probablyprofit import AnthropicAgent, PolymarketClient

# Connect to Polymarket
client = PolymarketClient(private_key="0x...")

# Create AI agent with your strategy
agent = AnthropicAgent(
    client=client,
    strategy_prompt="""
    Buy undervalued YES positions on high-volume
    political markets. Max $20 per trade.
    """
)

# Start trading
await agent.run_loop()
